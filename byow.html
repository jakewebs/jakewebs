About This Project

Language: Java

This project was done in my Data Structures & Algorithms class with a partner to apply our work in the class. Together, we made a world generation algorithm that would generate rooms of random sizes at random locations, and connected this with an intelligent hallway algorithm that opened doors to connect all the rooms. On the other hand, players could choose to create their own worlds with our Build option. Notably, the world generation algorithm is pseudorandom, so using the same seed twice would yield the same world.

With this world generated, we then chose random locations for the player, items, enemies, and even the world theme. Then, players would have to try and collect the Crown to gain the ability to vanquish the clever enemies, which prior to that powerup would chase the player to try and take away their lives (much like PAC-MAN). When the player either reached zero lives or defeated all the enemies, the game would be over. Additionally, players can save the game, quit, and load the game once again to keep playing from the same point.


My Contributions

  • Wrote an A* algorithm that was used by enemies to find the shortest path to the player, thus making the game action-packed.
  • Worked on the hallway generation algorithm, which used a QuickUnion to make sure all of the rooms were connected and kept working until they were.
  • Implemented many of the key game mechanics, from lives to powerups to player movements.
